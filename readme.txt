Easily create an organized Link To Us page on your WordPress site with linkable graphics, text and links.

== Description ==

Increase your Website traffic and page ranking with this simple but dynamic Link To Us WordPress plugin. WP Link To Us was created because we believe linking to you should be easy. This plugin will place a complete "link to us" page on your site with just a few clicks. Your users will find an organized list of your graphical assets, text and links in raw HTML format all available with a single click. 

WP Link To Us - Because linking to you should be easy!

== Installation ==

1. Extract the zip file into your plugins directory into its own folder (wplinktous).
2. Activate the plugin in the Plugin options.
3. Customize and add linkable assets to the "WP Link To Us" menu from the Settings panel.
4. Place the shortcode [wplinktous] in any of your Wordpress pages to publish your new WP Link To Us page

Enjoy and thank you for using WP Link To Us.

== Frequently Asked Questions ==

= Where are the settings for WP Link To Us? =

After a successful install, check under the Settings menu for WP Link To Us.

= What provides the one-click, copy-to-clipboard functionality? =

It's a small Flash file in the js folder.

= When I uninstall and remove the plugin, does that remove the wplinktous table in the database? =

Yes. If you reinstall, the previous data you entered will be gone.
